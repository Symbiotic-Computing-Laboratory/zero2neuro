# Zero2Neuro


Zero2Neuro is a no-code toolbox for constructing, training and
evaluating Deep Neural Network (DNN) models for a wide range of modeling
problems.  This toolkit allows the user to easily specify the details
of their experiments using small text-based files, including:

1. Loading data stored in a variety of formats, including CSV, XLSX,
and others;  configuring the data for use in DNN training and evaluation.

2. Creating of Deep Neural Network models from several flexible DNN
model schemata, including fully-connected networks (FCNs), convolutional
neural networks (CNNs), and recurrent neural networks (RNNs).  The
user specifies the structural details of their specific model.

3. Configuring a standard DNN training and evaluation engine.  This engine
supports the production of result reports in various formats,
including hooks for [Weights and Biases](https://wandb.ai).

Zero2Neuro can be used in different contexts, including:
1. The command line; this is particularly useful for using
supercomputers and other remote machines
2. Jupyter notebook
3. Within your own Python code
