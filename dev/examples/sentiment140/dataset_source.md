# Source of Dataset
This dataset is a subsampled dataset directly from https://www.kaggle.com/datasets/krishbaisoya/tweets-sentiment-analysis?resource=download.  
  
This dataset is a modified version of the Sentiment140 dataset which collected 1.6 million tweets from the Twitter API and assigned them a sentiment from 0-4. This modified version appears to clean up the messages to only the sentences and also simplifies 0-4 to 0-1. More about the original dataset can be found at the following places:  
http://www.sentiment140.com/  
https://www.tensorflow.org/datasets/catalog/sentiment140
